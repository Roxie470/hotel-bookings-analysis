## Overview
This project analysis explores the bookings of City and Resort hotels between 2015-2017 to uncover patterns, trends and insights that can support better hotel management decisions.

## Dataset
-Source: The dataset was provided by Quantum Analytics.NG. 
-Format: xlsv
-Description: This dataset contains 119390 observations for a City Hotel and a Resort Hotel. Each observation represents a hotel booking between the 1st of July 2015 and 31st of August 2017, including booking that effectively arrived and booking that were canceled. It contains 36 columns.
